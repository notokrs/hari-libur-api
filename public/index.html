Hari Libur & Hari Penting API

API untuk menampilkan hari libur dan hari penting lainnya

Respon API

{ 
	date: "2022-05-02", // yyyy-mm-dd 
	event: "Hari Raya Idul Fitri 1443 H",
	is_national_holiday: true 
}

Endpoint

Endpoint Fungsi
/api Menampilkan keseluruhan hari libur & hari penting ditahun sekarang.
/api?year=2022 Menampilkan keseluruhan hari libur & hari penting ditahun 2022.
/api?month=5&year=2022 Menampilkan keseluruhan hari libur & hari penting dibulan Mei tahun 2022.
Request endpoint selain daftar diatas dan tahun yang tidak ada dalam database akan menghasilkan array kosong / [ ].
